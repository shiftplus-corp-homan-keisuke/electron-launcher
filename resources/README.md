# らっこらんちゃー - アイコンリソース

このディレクトリに以下のアイコンファイルを配置してください。

## 必要なファイル

| ファイル名 | 説明 | 推奨サイズ |
|---|---|---|
| `icon.ico` | Windowsアプリアイコン | 256x256 |
| `icon.png` | アプリアイコン (PNG) | 256x256 |
| `tray-icon.png` | タスクトレイアイコン | 16x16 または 32x32 |
| `tray-icon@2x.png` | 高DPIタスクトレイアイコン | 32x32 または 64x64 |

## 注意事項

- `icon.ico` は `make` コマンドでのビルドに必要です
- アイコンファイルがない場合、アプリは起動しますがトレイアイコンが表示されません
- `.gitignore` でバイナリファイルは除外されています
